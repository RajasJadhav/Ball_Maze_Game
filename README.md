# Ball Maze Game
A 3D ball rolling maze game built in Unity C# as a learning project.

## Gameplay
Navigate the ball through the maze, collect all gems, and reach 
the goal zone within the time limit.

## What I built
- Player ball movement using Rigidbody physics and AddForce
- Camera follow system with fixed offset
- Collectibles with event-driven score system using C# Actions
- Win/lose conditions with GameManager state machine
- UI — score display, timer, win panel, lose panel
- Main menu scene with scene management
